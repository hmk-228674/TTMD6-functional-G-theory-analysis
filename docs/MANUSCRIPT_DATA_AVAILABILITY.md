## Submission-ready English

### Data Availability

TTMD6 is a publicly available third-party dataset associated with Zhang et al. [19]. The exact archive analysed in this study is available as publisher-hosted Supplementary Information at https://static-content.springer.com/esm/art%3A10.1038%2Fs41598-024-54150-5/MediaObjects/41598_2024_54150_MOESM1_ESM.rar. The publisher-hosted file contains 341,074,031 bytes (MD5: `1c9ce9cbf79dd35dd22f16a7199e2a8c`; SHA-256: `93d1b52a470f14b9dc0ba0600959bff921be891a3da1b71e609bd328224b354d`). We downloaded the publisher file in full and verified it byte-for-byte against the analysis input. The raw coordinate archive is not redistributed in this repository. A later Figshare v2 record [40] has a different byte count and was not analysed. The analysis code, input-identity checks, participant-code and exact-deduplication audits, 14-label mapping, functional-aggregation sensitivity, derived results, bootstrap diagnostics, figure source data and exact software specifications are publicly available at https://github.com/hmk-228674/TTMD6-functional-G-theory-analysis. The frozen release used for this manuscript is `v1.0.1`, archived at https://doi.org/10.5281/zenodo.21400718. The all-versions DOI is https://doi.org/10.5281/zenodo.21382966. The byte-identical raw archive must be obtained from the publisher link above.
