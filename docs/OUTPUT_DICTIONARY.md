# Output dictionary

## Core fields

| Field | Meaning |
|---|---|
| `action_id` | Archive action identifier 1–6 |
| `action_label` | Canonical English display label |
| `action_label_en` | Stable underscore-separated English slug |
| `integrated_sigma2_athlete` | $B_a$, integrated between-athlete variance component |
| `integrated_sigma2_trial` | $W_a$, integrated within-athlete trial variance component |
| `R_L2_m<n>` | $R_{L^2,a}(n)=B_a/(B_a+W_a/n)$ |
| `required_n_R_L2_80` | Minimum integer trial count reaching 0.80 |
| `required_n_R_L2_90` | Minimum integer trial count reaching 0.90 |
| `required_n_mean_pointwise_R_80` | Minimum integer trial count whose phase-mean pointwise reliability reaches 0.80 |
| `required_n_mean_pointwise_R_90` | Minimum integer trial count whose phase-mean pointwise reliability reaches 0.90 |
| `pointwise_relative_reliability_m<n>` | Nodewise relative-reliability curve at trial count $n$; not the primary integrated estimand |
| `mean_pointwise_reliability_m<n>` | Trapezoidal mean of nodewise ratios; reported only to distinguish it from the trace ratio |
| `boundary_fraction` | Fraction of phase nodes whose athlete variance estimate is on the non-negative boundary |

The pre-v1.0.1 field names `required_m_pointwise_G80` and
`required_m_pointwise_G90` were removed because the statistic is neither a
standard functional G coefficient nor indexed by a different design variable
from the trace-ratio threshold.  The v1.0.1 names use `n`, identify the
phase-mean pointwise construction explicitly, and retain the target after a
separating underscore.

## Reference-results directories

- `primary/`: complete balanced action-specific estimates, uncertainty summaries, raw-length exclusion, balanced subsampling, and REML self-tests.
- `sensitivity/`: structural-zero, fixed-13-joint, and continuous threshold summaries.
- `hampel/`: Hampel-type local-high-value sensitivity summaries.
- `fixed8/`: fixed eight-joint representation summaries.
- `global/`: contextual fixed-six-label decomposition and pointwise marginal intervals.
- `assumption_influence/`: archive order, AR(1) scenarios, registration, leave-one-out, and bootstrap diagnostics.
- `cohort_estimand/`: participant-code boundary, exact paired-record deduplication, 14-label mapping, functional aggregation, and cohort-bootstrap summaries.
- `reproduction/`: clean-run environment and hard-regression status.

The repository intentionally omits raw coordinates, cached NumPy arrays, and full bootstrap-draw files. They are regenerated by the single entry point; selected summaries and all figure source data are retained for rapid inspection.
